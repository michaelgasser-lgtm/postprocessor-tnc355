# postprocessor-tnc355
Development of a post-processor for FreeCad 1.0.2 to create CNC code for TNC355.
# Howto install:
-Each file has to be in the same folder as tnc355.py
-Set Path in FreeCad 1.0.2 Settings of the CAM-Workbench to this folder.
